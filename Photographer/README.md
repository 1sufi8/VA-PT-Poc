# Photographer – VulnHub

## Description
Proof of Concept (POC) penetration test performed on the Photographer VulnHub machine.

## Attack Summary
- Network and service enumeration using Nmap
- SMB enumeration
- Web application enumeration and exploitation (Koken CMS)
- Reverse shell execution
- Privilege escalation to root
